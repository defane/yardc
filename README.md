# yardc
Yet another remote desktop client

## Installation

    python3 setup.py install
    cp yardc.example.yaml ~/.config/yardc/yardc.yaml

## Usage

    yardc --help
    Usage: yardc [OPTIONS] COMMAND [ARGS]...

    Options:
      --help  Show this message and exit.

    Commands:
      encrypt_password
      hosts
      machines
      rdp
      start
      status
      stop
